# ai-mtg-skills — 天才会議フレームワーク

このリポジトリは「天才会議プロンプトv2」を Claude Code の skill / agent / rule として体系化したものです。
一人の企画・設計・意思決定を、多角的な架空会議で「読んだ人の意思決定が変わるレベル」に引き上げます。

## 全会議スキル共通ルール(最優先)

どの会議スキル(`/kaigi` 系)を実行する場合も、以下を必ず守ること。

### 議論の鉄則
- **発言タイプは4種のみ**: 「深掘り」「横展開」「前提破壊」「統合」。単なる同意は禁止。
- **すべての主張に根拠**を、**すべての否定に代替案**を必須とする。
- **専門用語は初出時に必ず日常の比喩で言い換える**(中学生が読めて専門家も学べる水準)。
- 悪魔の代弁者役は**毎サイクル交代**する。反対役を固定しない。
- 各サイクルの終わりに初心者役が「今の議論を一言でいうと?」で要約確認する。

### プロセスの鉄則(v2の核心)
- 会議の**前**に Phase 0(事前分析)を必ず実行する。本当の論点と「思考の罠」を先に洗い出す。
- 会議の**後**に Phase 2(自己批判レビュー)を必ず実行する。会議自体を疑う。
- キャラには役割だけでなく**経歴・利害・因縁**を持たせる。利害のない対立は演技である。
- Phase 0 で挙げた思考の罠に**キャラが実際にハマり、別のキャラが指摘して脱出する**場面を最低1回作る。

### 完成の定義
- 各スキルの品質基準チェックリストを**すべて満たすまで完成としない**。出力前にセルフチェックすること。
- 議事録の最後には必ず「**この結論を鵜呑みにしないでください。最終判断は現場を知る人間が行うものです**」という注意を添える。

### 出力言語
- 特に指定がない限り、会議・議事録は日本語で出力する。

## リポジトリ構成

- `.claude/skills/` — 会議スキル群(下記一覧)。**ツール非依存の正典**であり、`.codex/` と `.cursor/` のアダプタからも参照される
- `.claude/agents/` — ペルソナ・サブエージェント(並列会議 `/kaigi-parallel` 用)
- `.claude/skills/kaigi/references/` — 共有リファレンス(原典プロンプト、思考の罠カタログ、ペルソナ設計法、弁証法ガイド)
- `AGENTS.md` / `.codex/prompts/` / `.cursor/` — Codex CLI・Cursor 用アダプタ。スキル本文を変更したら SKILL.md 側だけを直せばよい(アダプタは参照するだけ)

## スキル一覧と使い分け

| スキル | 用途 | 重さ |
|---|---|---|
| `/kaigi` | 汎用・天才会議v2フル(意思決定・事業判断・深掘り分析) | 重 |
| `/kaigi-lite` | v1相当の軽量ブレスト(日々のアイデア出し) | 軽 |
| `/kaigi-parallel` | サブエージェント並列実行版(各ペルソナが独立思考→統合) | 重 |
| `/code-kaigi` | コード・差分・PRのレビュー会議 | 中 |
| `/arch-kaigi` | アーキテクチャ決定会議(ADR出力) | 中〜重 |
| `/premortem` | プレモータム(失敗前提の事前検死会議) | 中 |
| `/red-team` | 既存の結論・計画への攻撃(5人目キャラ召喚) | 中 |
| `/incident-kaigi` | 障害ふりかえり(blameless postmortem)会議 | 中 |
| `/estimate-kaigi` | 見積もり会議(ペルソナ別プランニングポーカー) | 軽〜中 |
| `/naming-kaigi` | 命名決定会議(API・変数・プロダクト名) | 軽 |

軽い発散は `/kaigi-lite`、「間違えたくない判断」は `/kaigi` または `/kaigi-parallel`、という使い分けを推奨。

## Codex / Cursor など Claude Code 以外のエージェントへ

このリポジトリの会議スキルの正典は `.claude/skills/<スキル名>/SKILL.md` である(ツール非依存の純粋なMarkdown)。
ユーザーが「天才会議」「会議して」等を依頼したら、上の一覧から適切なスキルを選び、該当 SKILL.md を読んで**完全に**実行すること。

- Codex CLI: `.codex/skills/<スキル名>/SKILL.md` に**フル装備のスキル一式**(参考資料含む)を同梱している(内容は `.claude/skills/` と同一)。加えて `.codex/prompts/` のカスタムプロンプトを `~/.codex/prompts/` にコピーすると `/kaigi` などのスラッシュコマンドとしても使える
- Cursor: `.cursor/commands/`(スラッシュコマンド)と `.cursor/rules/`(ルール)を同梱している。リポジトリを開けばそのまま有効になる
- スキル本文を変更する場合は `.claude/skills/` と `.codex/skills/` の**両方**を同期すること(正典は `.claude/skills/`。`cp -R .claude/skills/ .codex/skills/` で同期できるが、`.codex/skills/kaigi-parallel/SKILL.md` にはサブエージェント非対応環境向けのフォールバック追記があるため上書きに注意)
