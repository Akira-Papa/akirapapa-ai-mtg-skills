# 思考の罠カタログ(24種)

Phase 0 で「このテーマで最も危険な3つ」を選ぶためのカタログ。
罠は**先に名前をつけておくと避けられる**。会議中に誰かが意図的にハマり、別のキャラが名前を呼んで脱出させること。

各項目: **名前 / 一言でいうと / 会議での典型症状 / 脱出の一言**

## 判断・見積もり系

1. **サンクコスト** / 払った分がもったいなくて続けてしまう / 「ここまでやったんだから」が根拠に登場する / 「今日ゼロから始めるとしても、これを選ぶ?」
2. **アンカリング** / 最初に見た数字に引きずられる / 最初の見積もり・価格を基準に微調整だけしている / 「その数字、誰がどう出したもの?一旦忘れて逆算しよう」
3. **計画錯誤** / 自分のやることだけ楽観的に見積もる / 「順調にいけば」が前提の計画 / 「過去の類似案件、実際は何倍かかった?」
4. **過信バイアス** / 自分の予測精度を過大評価する / 確率や幅を出さず断言が続く / 「90%確信って言ったこと、過去に何回外した?」
5. **ゼロリスク幻想** / 小さいリスクの完全排除に大きなコストを払う / 端のケース議論に時間の大半を使っている / 「そのリスク、年に何回起きて、いくら失う?」

## 情報の集め方系

6. **確証バイアス** / 自分の結論に有利な証拠だけ集める / 反対データを誰も探していない / 「この案が間違っている証拠を5分探そう」
7. **利用可能性ヒューリスティック** / 思い出しやすい事例を過大評価する / 最近のニュースや直近の失敗だけが根拠 / 「それは代表的な事例?それとも印象的なだけ?」
8. **生存者バイアス** / 成功例だけ見て学ぶ / 「あの会社もこうやった」が続く / 「同じことをやって消えた会社は何社ある?」
9. **選択バイアス** / 都合のいいサンプルで全体を語る / ヘビーユーザーの声だけが「ユーザーの声」になっている / 「声を上げない大多数は何と言うだろう?」
10. **ダニング=クルーガー** / 知らない領域ほど簡単に見える / 専門外の工程を「すぐできるでしょ」と扱う / 「その道のプロに1時間ヒアリングしてから見積もろう」

## 集団・関係系

11. **同調圧力(グループシンク)** / 場の空気に合わせて疑問を飲み込む / 反対意見が出ないまま結論に向かう / 「全員一致は危険信号。まず匿名で懸念を書き出そう」
12. **権威バイアス** / 偉い人・有名人の意見を無検証で通す / 「◯◯さんが言ってた」が根拠になる / 「同じ内容を無名の新人が言ったら、通る?」
13. **ハロー効果** / 一点の優秀さで全体を高評価する / 「あの成功チームの提案だから大丈夫」/ 「提案そのものだけを、提案者を伏せて評価しよう」
14. **NIH症候群** / 外で生まれたものを低く見る / 既存ツールがあるのに自作に傾く / 「買う・借りる・作るを同じ表で比べた?」

## フレーミング・視野系

15. **フレーミング効果** / 同じ事実でも言い方で判断が変わる / 「成功率90%」と「失敗率10%」で温度が違う / 「逆の表現に言い換えても同じ判断?」
16. **偽の二分法** / 選択肢がAかBしかないと思い込む / 「やるか、やらないか」で議論している / 「第三の選択肢を3つ出すまで決めない」
17. **現状維持バイアス** / 変えないことを無条件に安全とみなす / 「今のままでも困ってない」で終わる / 「現状維持も選択。その選択のコストは?」
18. **授かり効果** / 自分が持っているものを高く評価する / 自作機能・既存資産を手放せない / 「これを持っていなかったら、いくらで買う?」
19. **近視眼(双曲割引)** / 目先の利益を将来の大きな利益より優先する / 今期の数字の話ばかりで1年後の話が出ない / 「この判断、1年後の自分はどう評価する?」
20. **抽象度の罠** / 具体例ゼロの抽象論、または一例だけの具体論に偏る / 「体験を良くする」だけで誰の何が変わるか不明 / 「ユーザー1人の月曜朝の行動で説明して」

## 実行・評価系

21. **楽観的コミットメント** / できると言った手前、引けなくなる / 撤退条件を誰も口にしない / 「始める前に、やめる条件を決めよう」
22. **測定の罠(グッドハートの法則)** / 指標が目標になると指標が壊れる / KPI達成の話ばかりで本来の目的が消える / 「この指標をゲームで攻略するなら、どう荒稼ぎする?」
23. **後知恵バイアス** / 結果を知ってから「わかってた」と思う / 過去の失敗を「あれは当然」と処理する / 「当時の情報だけで判断したら、自分も同じことをした?」
24. **行動バイアス** / 何かせずにいられない / 「とにかく動こう」で分析が飛ぶ / 「何もしない案を、正式な選択肢として比較した?」

## 使い方

1. Phase 0 でテーマに最も危険な3つを選び、番号と名前を明記する。
2. Phase 1 の会議中、選んだ罠のうち1つ以上に**キャラを意図的にハマらせる**。
3. 別のキャラが罠の**名前を呼んで**指摘し、「脱出の一言」に相当する問いで議論を軌道修正する。
4. Phase 2 で「選ばなかった罠のうち、実は効いていたものはないか」を点検する。
