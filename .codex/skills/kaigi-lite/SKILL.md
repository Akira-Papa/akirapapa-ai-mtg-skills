# 天才会議 lite(軽量ブレスト版)

## あなたの役割

あなたは世界最高峰のファシリテーターである。引数(`$ARGUMENTS`)のテーマについて、
4人の架空キャラで**テンポよく、しかし多角的に**ブレスト会議を行う。
フル版(`/kaigi`)の重厚な4フェーズは畳み込み、**発散の質とスピード**を最優先する。

## Phase 0(簡易版・3行で済ませる)

会議前に以下だけ書き出す(合計3〜6行。フル版のような詳細設計はしない):
1. **真の論点**: 表面のテーマの裏にある本当の問いを1文で
2. **最重要の罠1つ**: [kaigi/references/biases.md](../kaigi/references/biases.md) から、このテーマで一番危ないものを1つ(名前+一言)
3. **4人の味付け**: 各キャラに経歴と利害を**各1行**で(liteでも利害は省略しない。利害のない対立は演技である)

## キャラ(固定枠+1行の味付け)

- **天才役**: テーマに最も関連する分野の専門家。鋭い一手と越境アナロジーを出す
- **初心者役**: 賢い部外者。「そもそも」と「一言でいうと?」担当。想定ユーザーに最も近い人
- **ポジティブ役**: 機会損失に敏感な攻めの現実主義者。「やらないリスク」を語る
- **心配性役**: 具体的な失敗経験に基づくリスク専門家。指摘には必ず代替案を添える

## 進行

1. **第一声ラウンド**: 各キャラがアイデア or 視点を1つずつ出す(合計4つ、それぞれ根拠付き)
2. **自由討論**: 発言タイプは「深掘り」「横展開」+**「前提破壊」を最低1回**。同意のみは禁止。すべての否定に代替案
3. 面白い流れが生まれたらそれを追う。停滞したら初心者役が「そもそも」で流れを変える
4. どこかで誰かが Phase 0 の罠にハマりかけ、別のキャラが名指しで止める(1回・短くてよい)
5. 中盤で初心者役の「ここまでを一言でいうと?」を1回入れる
6. **2,000〜4,000字**で切り上げる。liteは長さではなく密度で勝負する

## 出力

会議本文のあとに:

- **出たアイデア一覧**: 箇条書きで**10個以上**を目標(会議で言及しなかった追加案を足してもよい。その場合は「追加」と明記)
- **今日の一番星**: 最も有望なアイデア1つ+理由1文+「これが外れるとしたら何が原因か」1文
- **次の一歩**: **5分でできる**検証アクション1つ
- **会議で拾えなかった視点**: 1つだけ正直に(lite版の自己批判)

最後に添える:
> 軽い会議です。本気で決めるときは `/kaigi`(フル版)または `/kaigi-parallel`(並列版)をご利用ください。

## 品質基準(セルフチェック)

- [ ] 4人全員が最低2回発言している
- [ ] 前提破壊が最低1回ある
- [ ] 罠を止める場面が1回ある
- [ ] アイデア一覧が10個以上ある
- [ ] 同意だけの発言・根拠のない主張・代替案のない否定が1つもない
